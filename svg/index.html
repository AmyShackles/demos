Notes

  • Feel free to contribute via GitHub.
  • "VO" is used as an abbreviation for "Voiceover".
  • Narrator testing was done on Win 10 OS build 16299.522.
  • For IE, the attribute focusable="false" should be added to the SVG element to ensure there's no double focus for links and buttons (implemented) and to ensure a decorative SVG is not focusable (not implemented).

Summary

  • Support for inline content is getting better; Voiceover most improved!
  • For links and buttons, all combinations except Narrator support aria-label when SVG used as link and button content.
  • For links and buttons, all combinations except Narrator support an IMG with an alt attribute as link and button content.
  • This use case passes all tests: Content | IMG | alt + role=img
  • JAWS 18 + IE 11 passed all tests! VO (Mac 10.14.2) + Safari (Mac 12.0.2) passed all but one.

Related Articles
