Notes

  • Nav element used as container of breadcrumb content. This is subjective; usage also may depend on the number of other nav elements on page.
  • Visible "You are here" text used and marked up as heading. Providing this text is subjective; same verbiage used as label on nav element (via aria-label).
  • Unordered list used for breadcrumb items. Some folks argue that *ordered* list is appropriate but I disagree.
  • Last item is not linked as it is the current page.
  • aria-current on current item (last in list).
