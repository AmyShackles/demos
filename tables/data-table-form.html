Accessible data table with form elements

Below is an example of an accessible data table containing form elements (two select dropdowns).

Notes:

  • Standard table markup is used such as caption and TH elements.
  • The inputs each contain an aria-labelledby attribute which points to the text in the row and column headers.
  • To ensure aria-labelledby functions in as many screen reader-browser combinations as possible, the referenced ID is on a span element within the header cell.
